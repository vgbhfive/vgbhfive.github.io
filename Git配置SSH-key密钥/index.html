Git配置SSH key密钥

Posted on 2019-02-17 Edited on 2020-02-05

Git作为一种分布式的代码管理工具，在github上的代码管理都需要在本地配置对应的SSH key密钥，这样才能推送代码。

1. 设置Git的username和email

2. 生成SSH key密钥

1. 查看创建密钥是否成功

如果没有此文件，则没有创建密钥成功。

2. 生成密钥

连续点击三下回车，这时密码为空。最后会出现id_rsa和id_rea.pub这两个文件。

3. 在github中添加密钥

1. 登陆github

使用你的Github账号和密码进行登录。

2.添加密钥

在右上角中找到设置，找到SSH keys，添加密钥。密钥位于本地的id_rsa文件中。

3. 本地测试添加是否成功

返回successfully时，那么就添加成功了。
