引入

本次将数据从 MySQL 备份到 ClickHouse 采用的是 MinervaDB ClickHouse MySQL Data Reader ，该项目基础版本来自于 Alitinity/clickhouse-mysql-data-reader。

安装

这里必须使用 pip3，并且最少需要 Python 3.4 版本。

优点和限制

  • 仅支持 Insert 事务。
  • 不支持 Delete 和 Update。
  • 不支持 DDL，MySQL 源端任何不兼容的修改都会破坏同步数据的流畅。
